# 📊 Exploratory Data Analysis (EDA)  

This repository contains Jupyter Notebook(s) that perform Exploratory Data Analysis (EDA) using Python and popular data analysis libraries such as Pandas, Seaborn, and Matplotlib. The goal of this project is to explore datasets, uncover patterns, and generate meaningful insights through visualization and statistical techniques.
